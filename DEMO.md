# E-Grade – How to run & demo script

## How to run

```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

## Demo script (1–2 minutes)

1. **Landing** – Show headline and the two entry points.
2. **Institution Login** – Click “Institution Login” → Dashboard with KPIs and charts.
3. **Test flow** – Go to “Test New Component”, fill the form, click “Run Diagnostic” → see result (Capability Matrix, Grade, Reusability). Click “Add to Internal Inventory” or “List on Marketplace” for success message.
4. **Inventory** – Open “Inventory”, show table/card view and filters.
5. **Marketplace** – From landing click “Marketplace”. Try search e.g. “GPIO only board” to see filtered/sorted results. Use filters (category, grade, reusability).
6. **Component detail** – Click “View Details” on a card → Capability Matrix, Similar Matches. Click “Reusability: X%” for explanation modal. “Add to Cart”, “Contact Seller”, “Request Bulk” (demo messages).
7. **Cart** – Open “Cart” in header → adjust quantity, “Place order (demo)” → success message (no payment).
8. **Seller** – “Seller Dashboard” → My Listings, Sales (empty), Inventory, Analytics.

No payment is processed; all actions are simulated for the hackathon demo.
